## Hublify

Hublify is an open-source web application designed to deliver visually stunning, interactive experiences for streamers and their audiences. Built with React and leveraging advanced WebGL effects, Hublify provides a rich set of customizable visual backgrounds and tools tailored for live streaming, scheduling, and community engagement.

### Key Features

- **Dynamic Visual Effects:** A diverse library of real-time, GPU-accelerated backgrounds (Aurora, Galaxy, Liquid Chrome, and more) to enhance your stream or web presence.
- **Streamer Tools:** Dedicated pages for managing streamers, schedules, and display settings, making it easy to organize and showcase streaming content.
- **Modern UI/UX:** Responsive design powered by Tailwind CSS and PostCSS, ensuring a seamless experience across devices.
- **Extensible Architecture:** Modular React components and custom hooks for easy integration and future expansion.
- **Open Source:** Actively maintained and open for community contributions, feedback, and feature requests.

### Project Structure

- `src/effects/` — Collection of unique visual effect components
- `src/pages/` — Streamer management, schedule, and display pages
- `src/hooks/` — Custom React hooks for WebGL and resource management
- `src/lib/` — Shared UI components and utilities
- `public/` — Static assets and entry HTML
- `server.js` — Node.js server for backend or SSR
- `tailwind.config.js`, `postcss.config.js` — Styling configuration

### Getting Started

To run Hublify locally, clone the repository and follow the setup instructions in the documentation. Contributions are welcome!
